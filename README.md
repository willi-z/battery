# Battery Simulation with Fenics

# 0. First Time

````
docker run -ti -v ${PWD}:/home/fenics/shared --name battery dolfinx/lab:stable
````

# 1. Starting
````
docker start battery
````

# 2. Stopping
````
docker stop new-project
````
